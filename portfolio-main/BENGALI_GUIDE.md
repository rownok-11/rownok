## ওয়েবসাইট চালানো

### ডেভেলপমেন্ট মোডে (লোকাল কম্পিউটারে):
```bash
# Dependencies ইনস্টল করুন
pnpm install

# ডেভেলপমেন্ট সার্ভার চালু করুন
pnpm run dev
```

ব্রাউজারে এই URL খুলুন: `http://localhost:5173`

### প্রোডাকশন বিল্ড তৈরি করুন:
```bash
pnpm run build
```

## সমস্যা সমাধান

### সমস্যা: অ্যাডমিন লগইন করতে পারছি না
**সমাধান:**
- Username এবং Password ঠিকভাবে লিখেছেন কি না চেক করুন
- Capital letter/Small letter সঠিক আছে কি না দেখুন
- Copy-paste করার সময় extra space না থাকে খেয়াল রাখুন

### সমস্যা: সাইবার থ্রেট ম্যাপ লোড হচ্ছে না
**সমাধান:**
- ইন্টারনেট কানেকশন চেক করুন
- পেজ রিফ্রেশ করুন (F5)
- অন্য ব্রাউজার ট্রাই করুন

### সমস্যা: ডেটা সেভ হচ্ছে না
**সমাধান:**
- ব্রাউজারের LocalStorage enabled আছে কি না চেক করুন
- Private/Incognito mode ব্যবহার করছেন না তো?

### সমস্যা: পেজ লোড হচ্ছে না
**সমাধান:**
```bash
# node_modules মুছে আবার ইনস্টল করুন
rm -rf node_modules
pnpm install
pnpm run dev
```
